func (g *Gateway) Watch(ctx context.Context) error {
    for ev := range g.sensors.Stream(ctx) {

    }

01  /  Tentang

Membangun untuk memahami, membongkar untuk menguasai.

Mahasiswa Teknologi Informasi di Institut Teknologi Sepuluh Nopember (ITS) Surabaya. Ketertarikan saya bermula dari keamanan siber: analisis buffer overflow, pengembangan exploit & tooling red-team eksperimental di Nim, hingga reverse engineering biner.

Di luar itu saya gemar membangun dari nol: framework HTTP di C++, MCP server untuk red teaming, sistem peringatan dini industri, hingga bot trading berbasis machine learning. Secara akademik saya aktif di praktikum Ethical Hacking, AI/ML, Big Data, dan Jaringan Komputer.

Saat ini saya mendalami Nim, Elixir, Assembly, Verilog, dan Solidity secara paralel, karena memahami bagaimana sistem bekerja di level terdalam adalah fondasi untuk membangunnya dengan benar.
